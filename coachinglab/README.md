# Coachinglab

Content resource for coaches — nonlinear pedagogy, constraints-led approach, external cues library, lesson plans. Finnish-first, English follows.

Working name only. Rename when a real brand name lands.

## Status

- **2026-05-26**: project initialized. Content-first; no site stack yet.

## Structure

- `README.md` — this file
- `cues.md` — external cues library (single growing document)
- `mechanics.md` — vetted mechanical conventions (block role assignment, etc.). Each entry sourced. Read before writing about mechanics in any post or lesson.
- `fieldstones.md` — raw material dump (Weinberg's fieldstone method): stories, anecdotes, observations from coaching. Stones get assembled into posts later.
- `posts/` — blog posts, created when the first one is ready
- `lessons/` — CLA / NLP lesson plans in Finnish (translated from `~/Dropbox/Lentopallo/Harjoitusmateriaalit/`)

Source lesson plans live in `~/Dropbox/Lentopallo/Harjoitusmateriaalit/`. As pieces get translated to Finnish for publication, the translations go here.

## Voice / tone (public-facing content)

Target reader: a smart coach friend who has never read a coaching theory book. Budding coaches, coaches stuck in conventional methods, curious parents, players considering coaching. **Do not assume any motor-learning vocabulary** — including "NLP," "CLA," "constraints," "external focus," "affordances," etc.

Rules for public posts (this README is internal — different rules apply here):

- **Plain Finnish.** No theoretical acronyms in the lead. Assume the reader doesn't know what "NLP" or "CLA" mean.
- **Show, don't theorize.** Vignettes, side-by-side examples, "try this and see what changes." Theory references are optional appendices.
- **Name a concept only after the reader has felt it.** A post can demonstrate external-focus questions working without ever using the term "external focus."
- **Academic terms at the very end, as a footnote** — for motivated readers who want to dig into the literature ("this is called X in English; see Renshaw / Davids / Wulf"). A bridge, not a gatekeeping vocabulary tax.
- **One big idea per post.** The full CLA-style lesson plan is the *climax* of a series, never the first piece.

## Theory background (internal — informs the project, doesn't surface as jargon)

Drawing from:

- **Ecological dynamics / nonlinear pedagogy** — Davids, Renshaw, Chow, Button
- **Constraints-led approach (CLA)** — Newell (1986) taxonomy; Renshaw, Davids et al.
- **Attentional focus / external cues** — Wulf
- **Practitioner perspective** — what does the theory actually mean in a session

Two related but distinct content types:

- **External cues**: phrases that direct attention to the *effect* of movement (the cues library)
- **Constraints**: task / environment / individual manipulations that shape what the system can do (lesson plans)

Both belong under the broader NLP umbrella but they're separate deliverables.
